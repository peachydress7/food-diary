## 🗄️ STEP 1：建立 Supabase 資料庫（約5分鐘）

1. 打開 https://supabase.com，點「Start your project」
2. 用 GitHub 或 Email 註冊帳號
3. 點「New project」，填入：
   - Project name：`chitji`（或任何名字）
   - Database Password：設一個密碼（記下來）
   - Region：選 **Southeast Asia (Singapore)**（離台灣最近）
4. 等待建立完成（約1分鐘）
5. 建立好後，點左側選單「**SQL Editor**」
6. 點「**New query**」
7. 把 `supabase_setup.sql` 的內容貼進去
8. 點右上角「**Run**」
9. 看到「Success」就完成了 ✅

### 取得 API Key
1. 點左側「**Project Settings**」（齒輪圖示）
2. 點「**API**」
3. 複製這兩個值（等等要用）：
   - **Project URL**：長得像 `https://xxxxx.supabase.co`
   - **anon public**：很長的一串字

---

## 🚀 STEP 2：部署到 Vercel（約5分鐘）

1. 打開 https://github.com，註冊帳號（免費）
2. 點右上角「**+**」→「New repository」
3. Repository name 填：`chitji`
4. 選「**Public**」
5. 點「**Create repository**」

### 上傳檔案到 GitHub
1. 在剛建立的 repository 頁面，點「**uploading an existing file**」
2. 把整個 `chitji` 資料夾裡的所有檔案拖進去
   （注意：要上傳資料夾裡的內容，不是資料夾本身）
3. 點「**Commit changes**」

### 連結 Vercel
1. 打開 https://vercel.com，點「Sign Up」
2. 選「**Continue with GitHub**」，授權登入
3. 點「**New Project**」
4. 找到你剛建的 `chitji` repository，點「**Import**」
5. 直接點「**Deploy**」（不用改設定）
6. 等待部署完成（約1分鐘）

---

## 🔑 STEP 3：設定環境變數（最重要！）

1. 在 Vercel 專案頁面，點上方「**Settings**」
2. 點左側「**Environment Variables**」
3. 新增第一個變數：
   - Name：`SUPABASE_URL`
   - Value：貼上剛才複製的 **Project URL**
   - 點「**Save**」
4. 新增第二個變數：
   - Name：`SUPABASE_ANON_KEY`
   - Value：貼上剛才複製的 **anon public**
   - 點「**Save**」
5. 點上方「**Deployments**」
6. 點最新的那一筆右邊的「**⋯**」→「**Redeploy**」
7. 等待完成 ✅

---

## 📱 STEP 4：加入 iPhone 主畫面

1. 在 Vercel 專案頁面找到你的網址（例如 `chitji.vercel.app`）
2. 用 iPhone 的 **Safari** 打開這個網址
3. 點底部的「**分享**」按鈕（方框加箭頭 ↑）
4. 往下滑找「**加入主畫面**」
5. 名字填「吃記」，點「**新增**」
6. 完成！桌面上就有圖示了 🎉

## ❓ 常見問題

**Q：資料會不見嗎？**
A：不會！資料存在 Supabase 雲端資料庫，永久保存。

**Q：免費嗎？**
A：完全免費。Supabase 免費方案有 500MB 空間，Vercel 免費方案無限部署。對一般家庭使用完全夠用。

**Q：網路斷線也能用嗎？**
A：可以！斷線時會自動切換到離線模式，資料暫存在手機，等網路恢復後可以手動備份還原。

**Q：資料安全嗎？**
A：Supabase 使用銀行等級的加密，資料安全。但建議不要記錄過於私密的資訊。
